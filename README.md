# Animated Dark Theme Hero Section

A full-screen hero section featuring a halftone dot-grid canvas animation with a green gradient overlay, smooth text reveal effects, and interactive mouse ripples.

## Features

- Halftone dot-grid canvas animation (Canvas 2D)
- Interactive mouse trail & ripple effects
- Sequential text reveal: fade-in + slide-up + blur (3s delay)
- Green gradient background with black bottom fade
- Frosted glass pill behind the headline
- 1920 × 1080 hero layout
